# Path of Chaz

## Overview

Path of Chaz is a game heaviliy inspired by (Path of Achra)[https://store.steampowered.com/app/2128270/Path_of_Achra/].

## Scope

The intention of this project is to reverse engineer the core mechanics of Path of Achra and create a new game with a similar feel

## Pillars

### From Achra
- one click gameplay
	- choose from different stylistic 'clicks'
		- attack, stand still, pray
		- deploy strategies across styles if you want
        - controls simple: tab, space, enter. or left click, right click, middle click.
- immergent strategy
	- win-conditions for your character change as run evolves
	- carrots and sticks:
		- an unexpected power-gain can pull the player.
		- an unexpected curse can push the player
- tiered character customization
	- IE: Race, Class, God
	- Each tier is a unique, incredible source of power and identity
	- you cannot win in champion select, just make it much easier for yourself if desired.

### New Ideas
- deeper character attachment and role-playing
	- longer, multi-session runs
	- each character needs some way of portraying identity
- a sense of grand strategy and planning
    - progression from lone infantry to an alexander the great
    - reaching new stages allows the player to inherit new territory and resources, but require them to make more important decisions
- death remains permanent, but has legacy systems for permanent progression
